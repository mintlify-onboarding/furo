title: "개요 (Overview)"
---

**Furo**를 사용하면 이메일, 패스워드 로그인부터 **`Google`**, **`Kakao`** 로그인 등 `OAuth` 프로토콜을 지원하는 어떠한 서비스든 쉽게 연동해서 로그인 시스템을 구성할 수 있습니다.

아래 그림은 내 서비스에 Furo를 적용했을 때 어떻게 로그인이 동작하는지를 보여주는 플로우 다이어그램입니다.

![](/images/guides/flow_diagram_using_furo.png)

Flow diagram using Furo

로그인과 관련된 모든 오퍼레이션을 **Furo**가 중간에서 대신 처리해주고 미리 지정한 **redirectURI**로 페이지를 이동시킵니다.

로그인 이후 부터는 유저의 데이터에 접근할 수 있게 되고 해당 데이터를 활용하여 서비스 페이지를 구성할 수 있습니다.
