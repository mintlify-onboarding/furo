이 가이드는 Furo를 통해 Google 로그인을 간편하게 구현하는 방법을 안내합니다.

## Google 로그인 활성화 

1. 1.  
[Furo Console](https://console.furo.one/projects) 소셜 로그인 탭을 활성화합니다.

2. 2.  
테스트용 **Google OAuth Client ID**를 Google Login 섹션의 **API Key**로 입력합니다.

3. 3.  
  
![](/images/getting-started/%E1%84%89%E1%85%B3%E1%84%8F%E1%85%B3%E1%84%85%E1%85%B5%E1%86%AB%E1%84%89%E1%85%A3%E1%86%BA%202022-10-24%20%E1%84%8B%E1%85%A9%E1%84%92%E1%85%AE%206.14.06.png)  
버튼을 눌러서 Google 로그인을 활성화시킵니다.

| Google OAuth Client ID                                                   |
| ------------------------------------------------------------------------ |
| 196548732042-8q1inmcmrf8mp0jc4fdi9l9si5vnpgjb.apps.googleusercontent.com |

해당 키는 **테스트 목적으로만 사용**되어야 하며 의도하지 않은 오류가 발생할 수 있습니다. 따라서 프로덕션 환경에서 App을 배포하거나 다수의 개발자가 테스트하는 개발 환경에서 App을 테스트하는 경우 [Google OAuth](/guides/social-login/google-oauth) 페이지를 참고하여 직접 발급받은 키를 사용하시기 바랍니다.

![](/images/getting-started/%E1%84%89%E1%85%B3%E1%84%8F%E1%85%B3%E1%84%85%E1%85%B5%E1%86%AB%E1%84%89%E1%85%A3%E1%86%BA%202022-11-01%20%E1%84%8B%E1%85%A9%E1%84%92%E1%85%AE%204.41.09.png)
