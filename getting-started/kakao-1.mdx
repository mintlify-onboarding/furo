이 가이드는 Furo를 통해 Kakao 로그인을 간편하게 구현하는 방법을 안내합니다.

## Kakao 로그인 활성화

1. [Furo Console](https://console.furo.one/projects) 소셜 로그인 탭을 활성화합니다.

2. 테스트용 **Kakao REST API Key**를 카카오 Login 섹션의 **API Key**로 입력합니다.

3. 버튼을 눌러서 카카오 로그인을 활성화시킵니다.

| Kakao REST API Key               |
| -------------------------------- |
| dc6398c27709ce49d3b922a35c886dff |

<Warning>
  해당 키는 **테스트 목적으로만 사용**되어야 하며 의도하지 않은 오류가 발생할 수
  있습니다. 따라서 프로덕션 환경에서 App을 배포하거나 다수의 개발자가 테스트하는
  개발 환경에서 App을 테스트하는 경우 [Kakao
  OAuth](/guides/social-login/kakao-oauth) 페이지를 참고하여 직접 발급받은 키를
  사용하시기 바랍니다.
</Warning>

<Info>
    테스트용 Kakao REST API Key에서 허용되는 Redirect Url은 다음과 같습니다.
    - "http://localhost:3000"
    - "https://auth.furo.one"
    로컬 테스트 시 **3000 포트 이외 환경에서는 테스트용 Key가 정상 작동 하지 않습니다.**
</Info>
