## Entity

| Name                 | Type   | Description                        |
| -------------------- | ------ | ---------------------------------- |
| workspace\_id        | String | Workspace ID                       |
| project\_id          | String | Project ID                         |
| name                 | String | Name of workspace                  |
| invitation\_url      | String |                                   |
| logo\_url            | String |                                   |
| owner\_id            | String | Workspace owner's user ID          |
| owner\_email         | String |                                   |
| created\_at          | Date   |                                   |
| modified\_at         | Date   |                                   |
| allowed\_domains     | Array  | Allowed domain names for workspace |
| description          | String | Description of workspace           |
| total\_member\_count | Number | Number of total member             |
