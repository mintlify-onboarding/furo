### Entity

| Name                                | Type       | Description                                |
| ----------------------------------- | ---------- | ------------------------------------------ |
| uid                                 | string     | 사용자 id (uuid)                              |
| email                               | string     | 이메일                                        |
| projectId                           | string     | 사용자의 프로젝트 ID                               |
| profile\_url                        | string     |                                           |
| preferred\_username                 | string     |                                           |
| thumbnail\_url                      | string     |                                           |
| display\_name                       | string     |                                           |
| name                                | object     | 이름                                         |
| about\_me                           | string     |                                           |
| current\_position                   | string     |                                           |
| sign\_in\_methods                   | object     | 로인 방ㅓㅂ                                     |
| social                              | string     | 가입 경로 ("furo", "google", "kakao", "naver") |
| verified                            | object     | 인증 여부                                      |
| teams                               | string\[\] | 소속 팀 id 리스트 (Furo의 User만 해당)               |
| [wallet](/data-types/user/wallet) | object     | Furo에서 생성된 Web3 지갑                         |
| address                             | string     | wallet의 address                            |
| created\_at                         | Date       | 사용자 회원생성 시                                 |
| modified\_at                        | Date       | 사용자 접속 시                                   |
| photos                              | Array      |                                           |
| urls                                | Array      |                                           |
| privacy                             | object     |                                           |
