## Base URL

인증 API는 HTTPS를 통해 제공됩니다. 

문서의모든 URL의 기본은 **https://api.furo.one**입니다.

## Postman 으로 API 테스트하기

Furo의 Postman에 아래 링크로 접속합니다.

[![](/images/api)Furo APIFuro API](https://postman.furo.one)

Postman에서 Furo의 public API들을 직접 사용해보실 수 있습니다.

![](/images/api/image.png)

접속하셔서 우측 상단의 **Run in Postman** 버튼을 누릅니다.

![](/images/api/image.png)

**Postman Web to Fork** 를 선택하셔서 Fork하여 테스트 하실 수 있습니다.
